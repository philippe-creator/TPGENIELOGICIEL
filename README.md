# TPGENIELOGICIEL

Ceci est le dépôt des travaux pratiques (TP) dans le cadre du module Génie Logiciel.

## 📚 Structure du Projet

Ce dépôt contient plusieurs TP couvrant différents concepts de génie logiciel :

- **POO/** - TP sur la Programmation Orientée Objet en Java
- **mon-projet-node/** - TP sur Node.js avec des fonctions arithmétiques de base
- **projetnotion/** - Projet de gestion de tâches (To-Do List)
- **docker/** - TP sur la conteneurisation avec Docker et Docker Compose

## 📋 Contenu des TP

### TP POO
Implémentation de concepts orientés objet en Java :
- Classes abstraites
- Héritage
- Polymorphisme
- Surcharge de méthodes

### TP Node.js
Projet Node.js démontrant :
- Création de fonctions réutilisables
- Exportation de modules
- Tests basiques

### Projet Notion
Application de gestion de tâches.

### TP Docker
Architecture conteneurisée avec :
- Frontend
- Base de données PostgreSQL
- Interface d'administration pgAdmin
- Réseau Docker

## 🚀 Utilisation

Chaque TP dispose de son propre fichier README.md avec des instructions spécifiques.

## 📄 Rapports

Chaque TP est accompagné d'un rapport détaillé (format PDF) expliquant la mise en œuvre et les concepts abordés.
